# Tanvox Technologies - Software Company Website

A modern, responsive software company website built with React.js and Tailwind CSS.

## Features

- **Modern Design**: Clean, professional, futuristic interface with dark theme
- **Smooth Animations**: Entrance animations, parallax scrolling, and hover effects
- **Responsive Layout**: Fully responsive design for desktop, tablet, and mobile
- **Interactive Elements**: Glowing buttons, card hover effects, and smooth transitions
- **Performance Optimized**: Lightweight and fast-loading with minimal dependencies

## Tech Stack

- **React.js 18.2.0** - Frontend framework
- **Tailwind CSS** - Utility-first CSS framework
- **No UI Libraries** - Custom components only
- **CSS Animations** - Pure CSS animations (no animation libraries)

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm start
   ```

3. **Open Browser**
   Navigate to `http://localhost:3000`

## Build for Production

```bash
npm run build
```
